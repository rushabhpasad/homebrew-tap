# homebrew-tap

Homebrew tap for [MD Editor](https://github.com/rushabhpasad/md-editor) — a lightweight, cross-platform desktop Markdown editor.

## Install

```bash
brew tap rushabhpasad/tap
brew install --cask md-editor
```

## Update

```bash
brew upgrade --cask md-editor
```

## Uninstall

```bash
brew uninstall --cask md-editor
brew untap rushabhpasad/tap
```

## Contents

| Cask | Description |
|---|---|
| `md-editor` | Lightweight cross-platform desktop Markdown editor with live split-pane preview |

## Cask updates

Cask files in this tap are updated automatically by a GitHub Actions workflow in the main [md-editor](https://github.com/rushabhpasad/md-editor) repository whenever a new release is published.
